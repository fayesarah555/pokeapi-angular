About Pokemon API Project

This project utilizes the Pokemon API to fetch and display information about various Pokemon species.

Features

  • Fetches Pokemon details such as name, height, weight, and types.
  • Displays Pokemon sprites (images) using data from the API.
  • Allows navigation between different Pokemon using Angular routing.

Technologies Used

  • Angular Framework
  • HTML, CSS (SCSS)
  • TypeScript
  • HttpClient Module for API requests

Project Structure

The project is structured with Angular components:

  • PokemonListComponent: Lists all available Pokemon.
  • PokemonDetailComponent: Displays details of a selected Pokemon.
  • PokemonService: Handles API requests to retrieve Pokemon data.

Getting Started

To run the project locally:

  1. Clone the repository from GitHub.
  2. Install Angular CLI if not already installed:

    To install Angular CLI, run the following command:

    npm install -g @angular/cli. .
  3. Navigate to the project directory and run npm install to install dependencies.
  4. Run ng serve for a dev server. Navigate to http://localhost:4200/ in your browser.
